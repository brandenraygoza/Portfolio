### 3U CubeSat Mass Model
This folder contains the **SolidWorks** CAD model for a 3U CubeSat. I designed this structure to hold modular mass blocks at specific locations (calculated using my **MATLAB** code found in the **[MATLAB directory](../../MATLAB)**) to accurately recreate a target moment of inertia tensor.
